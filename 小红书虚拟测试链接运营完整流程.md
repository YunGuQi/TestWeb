# 小红书虚拟测试链接运营完整流程

> 来源：老罗1对1私课群课程记录（2026年1月20日）
> 赛道：小红书虚拟电商 · 心理测试链接

---

## 总览：五步闭环模型

```
选赛道 → 搭账号 → 选品 → 造内容 → 出单收款
         ↑                              ↓
         └────────── 持续发帖 ───────────┘
```

## 第一步：选赛道

### 为什么选测试类？
- 虚拟资料赛道竞争激烈（简历/PPT/教案太卷），测试类相对蓝海
- 交付极简：卖一个网站链接，无物流、无售后
- 复购逻辑强：一个用户喜欢测试，会连续买多个品类

### 主要品类
| 品类 | 典型产品 |
|------|----------|
| 心理健康 | SCL-90、抑郁症测试、焦虑测试 |
| 人格性格 | MBTI 16型、动物人格、天赋测试 |
| 情感关系 | 恋爱占有欲、成人依恋模式、原生家庭 |
| 趣味测试 | 心理年龄、性压抑指数、哈利波特分院帽 |

## 第二步：搭建账号（全套4个号）

### 账号结构
```
主账号（开店 + 实名 + 上架商品 + 发帖）
    ├── 关联账号 A（不开店，绑主号店铺发帖）
    ├── 关联账号 B（同上）
    └── 关联账号 C（同上）
```
- 一人最多实名认证 **2个账号**，其余用家人实名
- 注册需要独立手机号（淘宝购买手机卡），一机一卡
- 发帖建议用 **数据流量**，避免同 WiFi 限流

### 2.1 注册小红书
1. 手机下载小红书 App
2. 选【其他登录方式】→【手机号登录】（不要用微信登录）
3. 注册时设置：
   - **性别**：女（私聊转化效果更好）
   - **年龄**：25-35 岁
   - **兴趣**：随便选 4 个，不影响推流

### 2.2 完善资料（必填4项）
| 项目 | 要求 |
|------|------|
| **头像** | 小动物 / 可爱卡通 / 女生真人 |
| **名字** | 带"测试"字眼，如"小王心理测试"、"心理测试加油站" |
| **简介** | 只需一句：「所有测试仅供娱乐，侵权联系删除」|
| **生日** | 填成年日期即可 |

### 2.3 实名认证（必须）
路径：我 → 设置 → 账号与安全 → 实名认证

> 重要：未实名账号发帖一段时间后会被隐藏限流（帖子+商品全部不显示），实名后解除。

> 技巧：若实名被旧号占用 → 新号认证时选"注销旧账号" → 注销原因选"实名被占用" → 实名立即释放（账号本身7天后注销，但实名当场可用）

### 2.4 开通店铺（仅主账号）
路径：创作者中心 → 全部服务 → 开通店铺 → 立即开店

- 类型：**个人店铺**
- 资质：上传身份证正反面
- 规模：选**小额零售**
- 保证金：**可先不交**，等有订单提现时再交；参加平台活动（如双11）才需要提前交

## 第三步：选品（找爆款测试）

### 方法一：灰豚数据（推荐）
官网：https://xhs.huitun.com/
- 企业会员可在闲鱼购买：1元/小时，8元/天
- 操作路径：商品分析 → 商品榜单 → 笔记商品榜
- 分类选：**文教服务**（虚拟资料主战场）
- 看**周榜**（本周新爆品）+ **月榜**（持续爆品）结合
- 选定商品后 → 点击笔记分析 → 找到绑定账号 → 跳转小红书主页（这就是对标账号）

### 方法二：小红书千帆后台
官网：https://ark.xiaohongshu.com/app-system/home
- 路径：商品 → 商机中心 → 潜力商品
- 分类选：**电子资料包**
- 重点看两个维度：
  - **社区热度最高**：说明用户主动在搜
  - **货源最紧缺**：说明供给跟不上需求
- **同时出现在两个榜**的商品 = 最佳目标选品

### 选品核心标准
- 销量趋势**稳定**（不是一时爆发）
- 在测试大类内**品类要丰富**，不要只卖一个品
- 周榜月榜都有排名的品优先

## 第四步：获取/制作测试链接

### 方法一：直接购买同行商品（参考用）
- 买来看别人的发货内容和网站形式
- **注意以下情况不能直接使用**：
  - 有授权码限制（一单一码，买家不能重复测）
  - 内容里含有其他店铺引流链接
  - 声明"本网站完全免费，倒卖请举报" → **有法律风险，绝对不用**

### 方法二：用 AI 自建测试网站（强烈推荐）
工具：**百度秒哒**
官网：https://www.miaoda.cn/

**优势**：
- 不需要编程，不需要购买服务器
- 一键生成可访问的公开网址
- 永久有效，无侵权风险
- 每天登录赠 100 积分，生成一个网站约耗 40 积分（每天可生成 2 个）

**操作步骤**：
1. 打开秒哒网站，在对话框输入提示词
2. AI 自动生成完整网站页面
3. 点击右上角【发布】（需实名认证）→ 生成可分享链接
4. 后续修改：在右下角对话框输入要求 → 点击更新

**基础提示词模板**：
```
帮我制作一个SCL-90综合焦虑症测试网站，
要求每一题回答完成后才能进入下一题，
全部问题回答完成后提供详细分析
```

**进阶提示词加料**：
- UI风格（如：治愈系紫色主题）
- 首页添加【小红书店铺】按钮，点击跳转到你的店铺链接
- 具体题目数量
- 结果报告样式

> 小红书店铺链接获取方式：手机端打开店铺页面 → 右上角分享 → 复制链接

### 5.2 商品图制作（二选一）

2. 尺寸选"电商首图"

**方案B：豆包 AI 生图**
1. 打开 https://www.doubao.com/ → 图像生成
2. 比例选 1:1
3. 提示词：`帮我生成一个商品首图，主题是"xxx测试"，商品里面的文字你可以自己设定`
4. 直接下载使用（水印无所谓）

### 5.3 商品类目

> ⚠️ **2026年2月新规更新**：原「电子资源」类目现需官方邀请入驻并提供原创证明，保证金升至1000元；乱挂靠类目会被封店。

**现行最稳妥方案**：上架到 **【艺术品及周边】→【艺术定制】→【虚拟定制】** 类目（需配合第三方自动发货工具）

上架步骤：
1. 进入千帆后台，选择【普通方式发布商品】
2. 填写名称、主图等基本信息
3. 在底部点击【继续丰富商品信息】
4. 运费模板选【全国包邮】，发货模式选【现货/24h】，售后选【不支持七天无理由退货】
5. 物流模板若要求填地址，虚拟商品随便填一个地址即可
6. 提交上架

### 5.4 商品标题
复制同行标题，改动几个词即可。例：

## 第六步：设置自动发货

> ⚠️ **2026年2月新规更新**：「虚拟定制」类目不支持千帆后台原生自动发货，需使用第三方工具 **阿奇索** 实现。

### 工具：阿奇索
官网：https://www.agiso.com/

**收费说明**：1个月30元，新用户有 **7天免费体验**，可先用免费期跑通流程。

### 操作步骤

1. 打开阿奇索网站 → 选择产品中心的【自动发货】
2. 点击【小红书发货】→ 扫码登录开了店铺的账号（或用千帆App扫码）
3. 点击左侧【商品设置】→ 点击【添加】，选择要设置自动发货的商品
4. 点击商品后面的【修改】，设置自动回复内容：
   - 勾选【自动更新物流状态】（自动确认发货）
   - 勾选【千帆消息】（通过客服消息发给买家，不要选短信）
   - 勾选【不使用模板】（只发自己输入的内容，不加多余废话）
5. 发货内容填入测试网站链接，例：`复制到浏览器打开，可多次重复测试：https://xxx`
6. 回到首页点击【助手下载】→ 下载自动发货助手到电脑 → 扫码登录店铺账号

> ⚠️ **重要**：自动发货助手需要**电脑保持开机运行**，关闭后无法自动发货。无条件保持开机的可租用云电脑（费用不高）。

### 发货链接来源
- **推荐**：用百度秒哒自建的测试链接（无侵权风险，可重复测试）
- **备选**：购买同行商品，用对方发货的链接——前提是自己先测试确认**可以多次重复测试**，有邀请码限制（一人一次）的不能用

## 第八步：发布笔记

### 基本规则
- **必须用手机发布**（电脑发不计入笔记权重）
- 每天最少发 **5条**，一机一卡用数据流量
- 同一 WiFi 下多号发帖容易限流
- **🔥 高阶打法：冷启动期“一天5篇错峰 A/B 测”**
  - **物理去重（安全护城河）**：新号暴力起号必须一天5更，但如果发高度相似的图片，会被系统判定为“恶意铺货”而降权。同一套测试题，必须通过代码或PS做物理去重（每套赋予不同的微小背景色偏、边距扰动、或覆盖隐形SVG噪点膜），打破系统的感知哈希（pHash）查重。
  - **错峰发布（流量最大化）**：去重后的 5 篇笔记绝对不能连发！必须分散卡在 5 个流量高峰期发布（例：08:00 早通勤, 12:30 午休, 15:30 摸鱼, 19:00 晚高峰, 22:00 深夜网抑云），确保每篇都能吃满初始流量池。
  - **赛马机制（去弱留强）**：发布 24 小时后看数据。如果某篇爆了（流量稳步上涨且出单），**立刻停止发新帖**，避免流量分流，让爆款飞满3-5天；如果某篇扑街（流量不过百），立刻设为**仅自己可见**，保持主页清爽。

### 笔记构成（6个要素）

#### 1. 封面图
- **方式**：小红书 App → 首页加号 → 写文字 → 输入标题文字 → 下一步 → 选自动生成的封面模板
- 直接抄对标账号的标题文字做封面，无需P图
- 封面风格：要么**纯文字**，要么**测试结果实拍截图**（两者选其一，保持统一）
- 不要：大量表情包、艺术字/花字、过多图案

#### 2. 标题
直接照抄对标账号标题，共3种有效公式：
| 类型 | 示例 |
|------|------|
| **提问式** | "正常人装抑郁能骗过医生吗？" / "你的动物塑是？" |
| **制造危机/悬疑** | "摆烂太久天赋回收走了" / "抑郁症最吓人的地方不是心慌手抖" |
| **极端案例反差** | "抑郁等级拉满是什么感觉" / "恋爱占有欲指数100" |

#### 3. 文案
1. 找对标账号的爆款帖子，复制其文案
2. 粘贴到 **豆包** 或 **Kimi**
3. 输入提示词：`学习这篇文案的内容以及风格，按照风格帮我生成10条相似的文案`
4. 一次生成10条，对应10篇笔记，只换封面和标题即可

#### 4. 标签
直接复制对标帖子的标签，不用自己想

#### 5. 评论区引导文案
直接复制对标帖子评论区的置顶文案（通常是引导去主页/购买的话术）

#### 6. 挂商品链接
发布前 → 添加组件 → 商品 → 选择对应商品 → 发布

## 第九步：找好的对标账号（筛选标准）

### 合格的对标账号必须满足：
| 条件 | 说明 |
|------|------|
| 粉丝数 < 5000 | 最好在2000以内，粉多说明他用了其他推流手段 |
| 每日更新 | 最新帖子是最近两天的 |
| 选品丰富 | 店铺持续上架新品，多品类，不只卖一个测试 |
| 封面规范 | 固定风格（纯文字或实拍），无艺术字表情包 |
| 内容有钩子 | 提问/悬疑/极端案例，不是科普鸡汤 |

### 不合格的对标账号（不要学）：
| 问题 | 原因 |
|------|------|
| 表情包+花字封面 | 视觉杂乱，用户看不到重点 |
| 内容全是鸡汤/科普 | 已知道的人不需要测，不知道的人直接划走 |
| 只卖一个品 | 爆单靠花钱推流，不适合我们模仿 |
| 粉丝超5000 | 已经靠别的方式起来了，不是我们要走的路 |

### 找对标账号的方法：
1. 小红书搜索品名 → 点击商品 → 按销量排序
2. 点击商品店铺 → 点击【更新了x篇笔记】→ 进入主页
3. 或通过灰豚数据 → 商品 → 笔记分析 → 绑定账号 → 跳转主页

## 风险提示

| 风险 | 规避方法 |
|------|----------|
