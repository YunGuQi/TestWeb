# 安安心灵自评馆 - 完整终版设计规范

## 1. 核心定位与合规
- **站名**：安安心灵自评馆 (An'an Soul Self-Assessment Hall)
- **定位**：专业自评、心灵探索（弱化“1分钱”廉价感）。
- **承诺**：1分钱=完整结果，无隐藏消费，无强制关注。
- **合规**：本测评仅供情绪状态自我参考，非医疗诊断。
- **转化**：结果页软引导6.9元全店打包。

## 2. 视觉风格 (Visual Identity)
- **主色调**：
  - **浅雾蓝** (`#D4E2EF` / `#B0C4DE`)：专业、冷静、治愈。
  - **柔粉白** (`#FFFAF0` / `#FFF5F5`)：温暖、包容。
- **文字**：系统默认黑体，深灰 (`#2D3748`) 主字，中灰 (`#718096`) 辅字。
- **风格**：高级不廉价，留白充足，卡片圆角适中 (`12px`)。
- **禁忌**：无红色爆炸贴、无闪烁动效、无“限时秒杀”字样。

## 3. 页面内容设计

### 第一页：入口页 (Landing Page)
- **品牌**：安安心灵自评馆
- **主标题**：情绪内耗·专业自评测评
- **副标题**：10题快速自测 | 即刻生成完整结果报告
- **信任栏**：✅ 完整结果直达 | ✅ 无隐藏消费 | ✅ 全程自助体验
- **按钮**：[ 开始自评 ] (柔和醒目)
- **底部**：本测评仅供情绪状态自我参考，非医疗诊断

### 第二页：测试页 (Quiz Page)
- **进度条**：当前进度：X/10题
- **题目**：10道新题（见下文）
- **选项**：4项 (A/B/C/D)
    - A·完全不符合 (1分)
    - B·偶尔符合 (2分)
    - C·比较符合 (3分)
    - D·完全符合 (4分)

### 第三页：结果页 (Result Page) - 采用“温柔共情版”
- **等级**：轻度内耗 / 中度内耗 / 重度内耗
- **解读**：(采用版本1文案结构)
    - 你习惯把情绪藏在心里，容易自我消耗
    - 对周遭感受敏锐，常因小事牵动情绪
    - 看似平静，内心却一直在反复纠结
- **建议**：
    - 每天留10分钟放空时间，不强迫自己思考
    - 接纳自己的情绪，不必事事追求完美
- **转化**：
    - 💌 更全面的自我探索
    - 推荐：高敏感人格｜身心压力状态｜MBTI人格｜社交疲惫度
    - 按钮：👉 小红书店铺「安安心理自评屋」 (6.9元解锁全店)

### 扩展：介绍页 (Intro Page)
- 保持现有功能，样式适配新视觉。

6. 总觉得心里装着事，无法真正放松下来

8. 习惯自我否定，觉得自己不够好
9. 睡前容易回想过往事，导致入睡困难
10. 面对选择时，总担心做出错误的决定
