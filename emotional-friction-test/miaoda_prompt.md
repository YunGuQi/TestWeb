# 秒哒AI设计提示词 - 心理测试网站

## 🎯 项目概述

设计一个**情绪摩擦测试**的心理评测网站，风格参考MBTI性格测试和16Personalities，采用温暖专业的紫色调，卡片化设计语言，适配桌面和移动端。

---

## 🎨 整体视觉风格

**主题词：** 疗愈系、专业感、温暖、现代、极简

**色彩方案：**
- 主色：紫罗兰色 #8B5CF6
- 深紫文字：#4C1D95
- 浅紫背景：#F8F5FF / #FAF5FF
- 纯白卡片：#FFFFFF
- 装饰浅紫：#E9D8FD

**设计语言：**
- 大圆角（16-24px）
- 柔和渐变背景
- 轻盈紫色阴影
- 卡片式布局
- 充足留白

---

## 📱 页面结构（共4个页面）

### 页面1：欢迎页 (Landing Page)

**布局：** 垂直居中单栏

**设计元素从上到下：**

1. **顶部留白区**（占8%高度）

2. **品牌标识**
   - 文字："安安心灵自评馆"
   - 字号：20px，灰色，居中
   - 字间距：1px

3. **主标题**
   - 文字："情绪摩擦测试"
   - 字号：28px，深紫色 #4C1D95，粗体
   - 居中对齐

4. **副标题标签**
   - 文字："了解你的情绪模式"
   - 样式：浅紫底色胶囊形状，圆角20px
   - 字号：14px，紫色文字
   - 宽度自适应，居中

5. **信任指标卡片**
   - 白色卡片，圆角16px
   - 内含3个横向排列的项目：
     * 图标：🔒 + "隐私保护"
     * 图标：⚡ + "5分钟快测"
     * 图标：📊 + "专业分析"
   - 每项图标在上、文字在下，垂直排列
   - 整体居中

6. **主按钮**
   - 文字："开始测试"
   - 全宽，高52px
   - 背景：紫色渐变或纯紫 #8B5CF6
   - 圆角50px（全圆角）
   - 白色文字，字重600
   - 紫色阴影向下扩散

7. **免责声明**
   - 小字：11px，浅灰色
   - 文字："本测试仅供参考，不构成专业诊断"
   - 居中

**背景：** 淡紫色径向渐变（中心#E9D8FD → 边缘#F8F5FF）

## 🎯 设计要点

### 响应式适配

**桌面端（≥768px）：**
- 主容器：1000px宽居中
- 容器白底，圆角24px
- 周围有紫色背景
- 高度85vh，阴影明显

**移动端（<768px）：**
- 全屏流式
- 背景透出淡紫色
- 卡片间距适当减小

### 交互细节

1. **所有按钮：**
   - 点击时缩小到96-98%
   - 过渡动画0.1s

2. **页面切换：**
   - 淡入淡出 + 轻微上滑
   - 时长0.4s

3. **进度条：**
   - 平滑过渡动画
   - 紫色填充

### 特殊组件

**未开发测试标识：**
- 价格后有红色圆圈感叹号
- 文字："测试未开发"，12px灰色
- 按钮文字改为："关注小红书"（小红书红色#FF2442）

## 🖼️ 视觉参考描述

**想象场景：** 打开网站就像进入一个温暖的心理咨询室，柔和的紫色灯光，干净整洁的白色家具，圆润的边角让人放松，专业而不冷漠。

**卡片质感：** 像悬浮在空中的轻盈纸片，有淡淡的紫色光晕阴影。

**按钮感觉：** 像软绵绵的紫色靠枕，点击下去会轻微回弹。

**整体氛围：** 现代心理健康应用的专业感 + Instagram小清新风格的视觉美感。
