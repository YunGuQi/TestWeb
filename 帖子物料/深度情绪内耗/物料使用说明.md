# 深度情绪内耗测试 - 物料使用说明

## 1. 文件夹结构说明
- `发帖文案与配置.md`：包含5个不同角度的帖子文案、标题和标签，用于矩阵号分发或赛马测试。
- `post_generator.html`：帖子图片全自动生成器。采用纯前端技术（html2canvas + File System Access API）实现。
- `千帆/`：用于存放千帆上架所需的极简主图及商品详情文案（在需要时另行生成）。

## 2. 如何使用自动化生成器 (post_generator.html)
1. 在浏览器（推荐使用 Chrome 或 Edge）中直接双击打开 `post_generator.html`。
2. 页面中会展示7张帖子的预览画板（1张首图 + 5张测试题 + 1张尾部引导图）。
3. 如果需要临时修改文案，可以直接在浏览器中按 `F12`，或者使用代码编辑器修改 `post_generator.html` 里的文字内容。
4. 点击页面顶部的 **“一键生成并下载所有图片”** 按钮。
5. 浏览器会弹出一个“选择文件夹”的窗口。请选择当前系统中的一个目录（建议在当前目录新建一个以今天日期命名的文件夹，如 `2026-06-21`），并授予浏览器写入权限。
6. 等待几秒钟，7张分辨率为 1080×1440 的高清 PNG 图片将被自动保存到你指定的文件夹中。

## 3. 发帖注意事项
- 发布的图片顺序：`0-首图.png` -> `1-题1.png` ~ `5-题5.png` -> `6-引导图.png`。
- 请从 `发帖文案与配置.md` 中挑选一套文案配合发帖。
- 引导用户在评论区留下选项（如 ABBBA 等），并通过大号或小号回复他们，引导去主页链接进行完整测试。

## 4. UI 设计说明
本套物料采用了 **“暗夜科技风”**（黑色底色配合赛博蓝/赛博紫发光效果），契合“情绪黑洞”和“深层探索”的心理学意境，通过冷色调增强诊断的专业感与神秘感。
